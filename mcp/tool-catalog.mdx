The Naive platform provides **54 tools** across **11 categories**. 8 are registered as [core tools](/mcp/core-tools) in the MCP context window; the remaining 46 are accessible via the [meta-tools](/mcp/meta-tools) discovery flow.

<Note>
  **Core** tools (marked ★) are callable directly. **Extended** tools require `naive_search → naive_tool_help → naive_execute`. See the [API reference](/api-reference) for full parameter documentation.
</Note>
