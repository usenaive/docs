---
title: "Check Integrations"
description: "List all available integrations across CMS platforms, Composio toolkits, and browser connections."
api: "GET /v1/integrations"
---

No parameters required.

<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/integrations \
  -H "Authorization: Bearer nv_sk_your_key_here"
```
</CodeGroup>

<ResponseField name="integrations" type="object">
  Integration groups: `cms` (Webflow, Ghost, Shopify, Framer), `composio` (100+ SaaS toolkits), and `connections` (browser integrations).
</ResponseField>

```json Example Response
{
  "integrations": {
    "cms": [
      {"id": "cms_1", "type": "webflow", "name": "My Webflow Site", "status": "connected"}
    ],
    "composio": [
      {"toolkit": "slack", "status": "connected"},
      {"toolkit": "github", "status": "available"}
    ],
    "connections": [
      {"id": "conn_1", "platform": "LinkedIn", "status": "connected"}
    ]
  }
}
```

Call this first to discover what's available. Then use [Get Integration Tools](/api-reference/integrations/get-integration-tools) to see available actions.
