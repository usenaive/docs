---
title: "Get Social Post"
description: "Get full details of a specific social media post by ID."
api: "GET /v1/social/posts/:postId"
---

<ParamField path="postId" type="string" required>
  Social post ID.
</ParamField>

<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/social/posts/sp_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/social/posts/sp_abc123",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/social/posts/sp_abc123", {
  headers: { "Authorization": "Bearer nv_sk_your_key_here" },
});
const data = await response.json();
```

```javascript JavaScript

```
