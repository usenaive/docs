<ParamField header="X-Company-Id" type="string" required>
  The company the goal belongs to.
</ParamField>

<ParamField path="id" type="string" required>
  The goal ID to update.
</ParamField>

<ParamField body="title" type="string">
  Updated goal title.
</ParamField>

<ParamField body="description" type="string">
  Updated goal description.
</ParamField>

<ParamField body="status" type="string">
  Updated status: `active`, `completed`, or `archived`.
</ParamField>

Only include the fields you want to change. Omitted fields are left unchanged.

<CodeGroup>
```bash curl
curl -X PATCH https://api.usenaive.ai/v1/goals/goal_abc123 \
  -H "Authorization: Bearer nv_pk_your_key_here" \
  -H "X-Company-Id: company_abc123" \
  -H "Content-Type: application/json" \
  -d '{
    "status": "completed"
  }'
```

```python Python
import requests

response = requests.patch(
    "https://api.usenaive.ai/v1/goals/goal_abc123",
    headers={
        "Authorization": "Bearer nv_pk_your_key_here",
        "X-Company-Id": "company_abc123",
    },
    json={
        "status": "completed",
    },
)
data = response.json()
```
