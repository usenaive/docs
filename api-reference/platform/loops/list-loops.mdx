---
title: "List Loops"
description: "List all loops for a company, with optional filters."
api: "GET /v1/loops"
---

<Info>This endpoint is **coming soon** and is not yet available for use.</Info>

<ParamField header="X-Company-Id" type="string" required>
  The company to list loops for.
</ParamField>

<ParamField query="employee_id" type="string">
  Filter loops assigned to a specific employee.
</ParamField>

<ParamField query="status" type="string">
  Filter by loop status. One of `running`, `paused`, or `completed`.
</ParamField>

<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/loops?status=running \
  -H "Authorization: Bearer nv_pk_your_key_here" \
  -H "X-Company-Id: company_abc123"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/loops",
    headers={
        "Authorization": "Bearer nv_pk_your_key_here",
        "X-Company-Id": "company_abc123",
    },
    params={"status": "running"},
)
data = response.json()
```

<ResponseField name="loops" type="array">
  Array of loop objects.
</ResponseField>

<ResponseField name="loops[].id" type="string">
  Unique identifier for the loop.
</ResponseField>

<ResponseField name="loops[].name" type="string">
  Display name of the loop.
</ResponseField>

<ResponseField name="loops[].mode" type="string">
  Loop mode: `recurring`, `reactive`, or `generational`.
</ResponseField>

<ResponseField name="loops[].program" type="string">
  Instructions executed each cycle.
</ResponseField>

<ResponseField name="loops[].cadence_seconds" type="number">
  Interval between cycles in seconds.
</ResponseField>

<ResponseField name="loops[].status" type="string">
  Current status: `running`, `paused`, or `completed`.
</ResponseField>

<ResponseField name="loops[].employee_id" type="string">
  The employee assigned to execute this loop.
</ResponseField>

<ResponseField name="loops[].last_cycle_at" type="string">
  ISO 8601 timestamp of the last completed cycle, or `null`.
</ResponseField>
