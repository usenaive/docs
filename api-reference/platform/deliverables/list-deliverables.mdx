---
title: "List Deliverables"
description: "List all deliverables for a company, with optional filters."
api: "GET /v1/deliverables"
---

<Info>This endpoint is **coming soon** and is not yet available for use.</Info>

<ParamField header="X-Company-Id" type="string" required>
  The company to list deliverables for.
</ParamField>

<ParamField query="status" type="string">
  Filter by review status. One of `pending_review`, `approved`, or `rejected`.
</ParamField>

<ParamField query="task_id" type="string">
  Filter deliverables linked to a specific task.
</ParamField>

<ParamField query="type" type="string">
  Filter by deliverable type (e.g. `blog_post`, `email_draft`).
</ParamField>

<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/deliverables?status=pending_review \
  -H "Authorization: Bearer nv_pk_your_key_here" \
  -H "X-Company-Id: company_abc123"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/deliverables",
    headers={
        "Authorization": "Bearer nv_pk_your_key_here",
        "X-Company-Id": "company_abc123",
    },
    params={"status": "pending_review"},
)
data = response.json()
```

<ResponseField name="deliverables" type="array">
  Array of deliverable objects.
</ResponseField>

<ResponseField name="deliverables[].id" type="string">
  Unique identifier for the deliverable.
</ResponseField>

<ResponseField name="deliverables[].title" type="string">
  Display title of the deliverable.
</ResponseField>

<ResponseField name="deliverables[].type" type="string">
  Deliverable type (e.g. `email_draft`, `blog_post`, `social_post`).
</ResponseField>

<ResponseField name="deliverables[].folder" type="string">
  Folder category the deliverable belongs to.
</ResponseField>

<ResponseField name="deliverables[].status" type="string">
  Review status: `pending_review`, `approved`, or `rejected`.
</ResponseField>

<ResponseField name="deliverables[].content" type="string">
  Truncated content preview (first 200 characters).
</ResponseField>

<ResponseField name="deliverables[].task_id" type="string">
  ID of the task that produced this deliverable, or `null`.
</ResponseField>

<ResponseField name="deliverables[].campaign_id" type="string">
  ID of the associated campaign, or `null`.
</ResponseField>

<ResponseField name="deliverables[].created_at" type="string">
  ISO 8601 timestamp of when the deliverable was created.
</ResponseField>
