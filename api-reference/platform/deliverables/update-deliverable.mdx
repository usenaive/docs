</ParamField>

<ParamField path="id" type="string" required>
  The deliverable ID to update.
</ParamField>

<ParamField body="title" type="string">
  Updated display title.
</ParamField>

<ParamField body="content" type="string">
  Updated deliverable content.
</ParamField>

<ParamField body="preview" type="string">
  Updated preview URL or HTML snippet.
</ParamField>

<ParamField body="file_path" type="string">
  Updated file or asset path.
</ParamField>

<CodeGroup>
```bash curl
curl -X PATCH https://api.usenaive.ai/v1/deliverables/del_abc123 \
  -H "Authorization: Bearer nv_pk_your_key_here" \
  -H "X-Company-Id: company_abc123" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Weekly Newsletter Draft v2",
    "content": "Updated content with latest metrics and customer stories."
  }'
```

```python Python
import requests

response = requests.patch(
    "https://api.usenaive.ai/v1/deliverables/del_abc123",
    headers={
        "Authorization": "Bearer nv_pk_your_key_here",
        "X-Company-Id": "company_abc123",
    },
    json={
        "title": "Weekly Newsletter Draft v2",
        "content": "Updated content with latest metrics and customer stories.",
    },
)
data = response.json()
```

```typescript TypeScript
const response = await fetch(
  "https://api.usenaive.ai/v1/deliverables/del_abc123",
  {
    method: "PATCH",
    headers: {
      "Authorization": "Bearer nv_pk_your_key_here",
      "X-Company-Id": "company_abc123",
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      title: "Weekly Newsletter Draft v2",
      content: "Updated content with latest metrics and customer stories.",
    }),
  }
);
const data = await response.json();
```
