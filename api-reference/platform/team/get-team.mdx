<Info>This endpoint is **coming soon** and is not yet available for use.</Info>

<ParamField header="X-Company-Id" type="string" required>
  The company to get the team roster for.
</ParamField>

Returns the operational view of the team roster with reporting structure for org-chart rendering. For management operations, use the [Employees endpoints](/api-reference/platform/employees/list-employees).

<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/team \
  -H "Authorization: Bearer nv_pk_your_key_here" \
  -H "X-Company-Id: company_abc123"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/team",
    headers={
        "Authorization": "Bearer nv_pk_your_key_here",
        "X-Company-Id": "company_abc123",
    },
)
data = response.json()
```

<ResponseField name="team" type="array">
  Array of team member objects.
</ResponseField>

<ResponseField name="team[].paperclipId" type="string">
  Unique Naive identifier for the team member.
</ResponseField>

<ResponseField name="team[].agentId" type="string">
  Internal agent identifier.
</ResponseField>

<ResponseField name="team[].name" type="string">
  Team member display name.
</ResponseField>

<ResponseField name="team[].title" type="string">
  Job title (e.g., "Content Writer").
</ResponseField>

<ResponseField name="team[].role" type="string">
  Role slug describing the team member's function.
</ResponseField>

<ResponseField name="team[].capabilities" type="array">
  List of primitive categories the team member can use.
</ResponseField>

<ResponseField name="team[].reportsTo" type="string">
  `agentId` of the team member's manager, or `null` if top-level.
</ResponseField>

<ResponseField name="team[].status" type="string">
  Current status: `active`, `paused`, or `decommissioned`.
</ResponseField>
