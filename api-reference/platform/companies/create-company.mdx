title: "Create Company"
description: "Create a new company under the authenticated user's account."
api: "POST /v1/companies"
---

<Info>
  This endpoint is **coming soon** and is not yet available for use.
</Info>

<ParamField body="name" type="string" required>
  Display name for the new company.
</ParamField>

<Warning>
  This is a **billable action**. If your plan's company limit has been reached, the API will return a `402` status with a Stripe payment link to upgrade.
</Warning>

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/companies \
  -H "Authorization: Bearer nv_pk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Acme Corp"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.usenaive.ai/v1/companies",
    headers={"Authorization": "Bearer nv_pk_your_key_here"},
    json={
        "name": "Acme Corp",
    },
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/companies", {
  method: "POST",
  headers: {
    "Authorization": "Bearer nv_pk_your_key_here",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    name: "Acme Corp",
  }),
});
const data = await response.json();
```
</CodeGroup>

<ResponseField name="id" type="string">
  Unique identifier for the newly created company.
</ResponseField>

<ResponseField name="name" type="string">
  Company display name.
</ResponseField>

<ResponseField name="slug" type="string">
  Auto-generated URL-safe slug derived from the company name.
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of when the company was created.
</ResponseField>

```json Example Response
{
  "id": "comp_abc123",
  "name": "Acme Corp",
  "slug": "acme-corp",
  "created_at": "2026-04-29T12:00:00Z"
}
```

**Error codes:**
- `402` — Plan company limit reached. Response body includes a `payment_url` to upgrade.
- `409` — A company with that name already exists on your account.
