<CodeGroup>
```bash curl
curl -X GET https://api.usenaive.ai/v1/companies/comp_abc123 \
  -H "Authorization: Bearer nv_pk_your_key_here" \
  -H "X-Company-Id: comp_abc123"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/companies/comp_abc123",
    headers={
        "Authorization": "Bearer nv_pk_your_key_here",
        "X-Company-Id": "comp_abc123",
    },
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/companies/comp_abc123", {
  method: "GET",
  headers: {
    "Authorization": "Bearer nv_pk_your_key_here",
    "X-Company-Id": "comp_abc123",
  },
});
const data = await response.json();
```

  Number of active employees (agents) in the company.
