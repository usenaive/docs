<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/employees/emp_abc123 \
  -H "Authorization: Bearer nv_pk_your_key_here" \
  -H "X-Company-Id: company_abc123"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/employees/emp_abc123",
    headers={
        "Authorization": "Bearer nv_pk_your_key_here",
        "X-Company-Id": "company_abc123",
    },
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/employees/emp_abc123", {

  },

const data = await response.json();
```

  Display name.
</ResponseField>

<ResponseField name="title" type="string">
  Job title.
</ResponseField>

<ResponseField name="role" type="string">
  Role slug.
</ResponseField>

<ResponseField name="email" type="string">
  The employee's provisioned email address (e.g., `alice@yourcompany.usenaive.ai`).
</ResponseField>

<ResponseField name="focus" type="string">
  Current focus area or priority directive.
</ResponseField>

<ResponseField name="context" type="string">
  Background context provided to the employee for decision-making.
</ResponseField>

<ResponseField name="strategy" type="string">
  High-level strategy the employee should follow.
</ResponseField>

<ResponseField name="notes" type="string">
  Operator notes attached to the employee.
</ResponseField>

<ResponseField name="memo" type="string">
  Extended internal memo (up to 4000 characters).
</ResponseField>

<ResponseField name="template" type="string">
  Template slug this employee was created from, or `null`.
</ResponseField>

<ResponseField name="status" type="string">
  Current status: `active`, `paused`, or `decommissioned`.
</ResponseField>
