<ParamField body="title" type="string" required>
  Short title describing the task.
</ParamField>

<ParamField body="description" type="string" required>
  Detailed instructions for the task.
</ParamField>

<ParamField body="assignee_employee_id" type="string" required>
  Employee ID of the agent that should execute the task.
</ParamField>

<ParamField body="priority" type="string">
  Priority level. One of `low`, `medium`, `high`, `urgent`. Defaults to `medium`.
</ParamField>

<ParamField body="goal_id" type="string">
  Associate the task with an existing goal.
</ParamField>

<ParamField body="campaign_id" type="string">
  Associate the task with an existing campaign.
</ParamField>

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/tasks \
  -H "Authorization: Bearer nv_pk_your_key_here" \
  -H "X-Company-Id: company_abc123" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Draft Q3 blog post",
    "description": "Write a 1,200-word blog post covering Q3 product updates.",
    "assignee_employee_id": "emp_writer01",
    "priority": "high",
    "goal_id": "goal_content_q3"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.usenaive.ai/v1/tasks",
    headers={
        "Authorization": "Bearer nv_pk_your_key_here",
        "X-Company-Id": "company_abc123",
    },
    json={
        "title": "Draft Q3 blog post",
        "description": "Write a 1,200-word blog post covering Q3 product updates.",
        "assignee_employee_id": "emp_writer01",
        "priority": "high",
        "goal_id": "goal_content_q3",
    },
)
data = response.json()
```
