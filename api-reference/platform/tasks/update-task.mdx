title: "Update Task"
description: "Update a task's status, priority, description, or add a comment."
api: "PATCH /v1/tasks/{id}"
---

  This endpoint is **coming soon** and is not yet available for use.

<ParamField path="id" type="string" required>
  Unique task identifier.
</ParamField>

<ParamField body="status" type="string">
  New status. One of `queued`, `in_progress`, `in_review`, `done`, `blocked`.
</ParamField>

<ParamField body="priority" type="string">
  New priority. One of `low`, `medium`, `high`, `urgent`.
</ParamField>

<ParamField body="description" type="string">
  Updated task description.
</ParamField>

<ParamField body="comment" type="string">
  Adds a comment explaining the change.
</ParamField>

<CodeGroup>
```bash curl
curl -X PATCH https://api.usenaive.ai/v1/tasks/task_001 \
  -H "Authorization: Bearer nv_pk_your_key_here" \
  -H "X-Company-Id: company_abc123" \
  -H "Content-Type: application/json" \
  -d '{
    "status": "in_review",
    "comment": "Draft completed, moving to review."
  }'
```

```python Python
import requests

response = requests.patch(
    "https://api.usenaive.ai/v1/tasks/task_001",
    headers={
        "Authorization": "Bearer nv_pk_your_key_here",
        "X-Company-Id": "company_abc123",
    },
    json={
        "status": "in_review",
        "comment": "Draft completed, moving to review.",
    },
)
data = response.json()
```
