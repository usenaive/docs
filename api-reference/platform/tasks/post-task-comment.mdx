<ParamField path="id" type="string" required>
  Unique task identifier.
</ParamField>

<ParamField body="content" type="string" required>
  The comment body text.
</ParamField>

<ParamField body="employee_id" type="string" required>
  ID of the employee or user posting the comment.
</ParamField>

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/tasks/task_001/comments \
  -H "Authorization: Bearer nv_pk_your_key_here" \
  -H "X-Company-Id: company_abc123" \
  -H "Content-Type: application/json" \
  -d '{
    "content": "First draft is ready for review.",
    "employee_id": "emp_writer01"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.usenaive.ai/v1/tasks/task_001/comments",
    headers={
        "Authorization": "Bearer nv_pk_your_key_here",
        "X-Company-Id": "company_abc123",
    },
    json={
        "content": "First draft is ready for review.",
        "employee_id": "emp_writer01",
    },
)
data = response.json()
```
