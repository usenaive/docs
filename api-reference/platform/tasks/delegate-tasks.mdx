---
title: "Delegate Tasks"
description: "Create and assign multiple tasks in a single batch request."
api: "POST /v1/tasks/delegate"
---

<Info>
  This endpoint is **coming soon** and is not yet available for use.
</Info>

<ParamField body="label" type="string">
  Optional human-readable label for this batch (e.g. `"Q3 content sprint"`).
</ParamField>

<ParamField body="batch_id" type="string">
  Optional client-supplied idempotency key for the batch.
</ParamField>

<ParamField body="tasks" type="array" required>
  Array of task objects to create. Each object supports the following fields:
</ParamField>

<ParamField body="tasks[].title" type="string" required>
  Short title describing the task.
</ParamField>

<ParamField body="tasks[].description" type="string" required>
  Detailed instructions for the task.
</ParamField>

<ParamField body="tasks[].assignee_employee_id" type="string" required>
  Employee ID of the agent that should execute the task.
</ParamField>

<ParamField body="tasks[].priority" type="string">
  Priority level. One of `low`, `medium`, `high`, `urgent`.
</ParamField>

<ParamField body="tasks[].goal_id" type="string">
  Associate the task with an existing goal.
</ParamField>

<ParamField body="tasks[].campaign_id" type="string">
  Associate the task with an existing campaign.
</ParamField>

```bash curl
curl -X POST https://api.usenaive.ai/v1/tasks/delegate \
  -H "Authorization: Bearer nv_pk_your_key_here" \
  -H "X-Company-Id: company_abc123" \
  -H "Content-Type: application/json" \
  -d '{

    "tasks": [
      {

      }
    ]
  }'
```

```python Python
import requests

    "https://api.usenaive.ai/v1/tasks/delegate",

        "Authorization": "Bearer nv_pk_your_key_here",

    },
    json={
        "label": "Q3 content sprint",
        "tasks": [
```

<ResponseField name="ok" type="boolean">
  `true` if all tasks were created.
</ResponseField>

<ResponseField name="tasks" type="array">
  Array of the newly created task objects.
</ResponseField>

<ResponseField name="tasks[].id" type="string">
  Unique task identifier.
</ResponseField>

<ResponseField name="tasks[].title" type="string">
  Task title.
</ResponseField>

<ResponseField name="tasks[].status" type="string">
  Initial status (always `queued`).
</ResponseField>

<ResponseField name="tasks[].assignee" type="string">
  Assigned employee ID.
</ResponseField>

<ResponseField name="tasks[].priority" type="string">
  Priority level.
</ResponseField>

<ResponseField name="tasks[].created_at" type="string">
  ISO 8601 timestamp.
</ResponseField>
