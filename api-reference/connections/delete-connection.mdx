---
title: "Delete Connection"
description: "Delete a browser integration connection and its stored credentials."
api: "DELETE /v1/connections/:connectionId"
---

<ParamField path="connectionId" type="string" required>
  Connection ID.
</ParamField>

<CodeGroup>
```bash curl
curl -X DELETE https://api.usenaive.ai/v1/connections/conn_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here"
```
</CodeGroup>

<ResponseField name="ok" type="boolean">
  Success flag.
</ResponseField>

```json Example Response
{
  "ok": true
}
```

<Warning>This permanently deletes the connection and all stored credentials.</Warning>
