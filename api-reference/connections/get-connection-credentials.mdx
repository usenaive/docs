---
title: "Get Connection Credentials"
description: "Get decrypted login credentials for a browser integration."
api: "GET /v1/connections/:connectionId/credentials"
---

<ParamField path="connectionId" type="string" required>
  Connection ID from [List Connections](/api-reference/connections/list-connections).
</ParamField>

<Warning>
  **Security:** Credentials are decrypted server-side. Only use them in browser automation contexts. Never display to users or log in plaintext.
</Warning>

<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/connections/conn_abc123/credentials \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/connections/conn_abc123/credentials",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/connections/conn_abc123/credentials", {
  headers: { "Authorization": "Bearer nv_sk_your_key_here" },
});
const data = await response.json();
```

```javascript JavaScript
const response = await fetch("https://api.usenaive.ai/v1/connections/conn_abc123/credentials", {
  headers: { "Authorization": "Bearer nv_sk_your_key_here" },
});
const data = await response.json();
```
</CodeGroup>

<ResponseField name="id" type="string">Connection ID.</ResponseField>
<ResponseField name="login_url" type="string">Login page URL.</ResponseField>
<ResponseField name="has_2fa" type="boolean">Whether 2FA is required.</ResponseField>
<ResponseField name="instructions" type="string">Special login instructions.</ResponseField>
<ResponseField name="username" type="string">Decrypted username.</ResponseField>
<ResponseField name="password" type="string">Decrypted password.</ResponseField>

```json Example Response
{
  "id": "conn_abc123",
  "login_url": "https://github.com/login",
  "has_2fa": false,
  "instructions": "Use SSO button if available",
  "username": "agent@acme.com",
  "password": "decrypted_password"
}
```
