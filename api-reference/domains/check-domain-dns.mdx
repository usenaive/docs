---
title: "Check Domain DNS"
description: "Check DNS propagation and Resend verification status for a domain."
api: "GET /v1/domains/:domainId/dns"
---

<ParamField path="domainId" type="string" required>
  Domain ID from [Register Domain](/api-reference/domains/register-domain) or [List Domains](/api-reference/domains/list-company-domains).
</ParamField>

<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/domains/dom_abc123/dns \
  -H "Authorization: Bearer nv_sk_your_key_here"
```
</CodeGroup>

<ResponseField name="verified" type="boolean">
  Whether DNS is fully verified.
</ResponseField>

<ResponseField name="records" type="array">
  DNS records. Each contains `type`, `name`, `value`, and `status`.
</ResponseField>

```json Example Response
{
  "verified": false,
  "records": [
    {"type": "MX", "name": "acme.com", "value": "feedback-smtp.us-east-1.amazonses.com", "status": "verified"},
    {"type": "TXT", "name": "acme.com", "value": "v=spf1 include:amazonses.com ~all", "status": "pending"}
  ]
}
```
