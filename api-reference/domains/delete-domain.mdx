---
title: "Delete Domain"
description: "Permanently remove a registered domain and all associated configuration."
api: "DELETE /v1/domains/:domainId"
---

<ParamField path="domainId" type="string" required>
  Domain ID.
</ParamField>

<CodeGroup>
```bash curl
curl -X DELETE https://api.usenaive.ai/v1/domains/dom_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.delete(
    "https://api.usenaive.ai/v1/domains/dom_abc123",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/domains/dom_abc123", {
  method: "DELETE",
  headers: { "Authorization": "Bearer nv_sk_your_key_here" },
});
const data = await response.json();
```

```javascript JavaScript
const response = await fetch("https://api.usenaive.ai/v1/domains/dom_abc123", {

```
</CodeGroup>

  Success flag.

<Warning>This is irreversible. DNS records, email routing, and hosting configuration are permanently removed.</Warning>
