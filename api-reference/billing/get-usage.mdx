---
title: "Get Usage"
description: "Get credit usage and billing information."
api: "GET /v1/usage"
---

<ParamField query="days" type="number">
  Number of days of usage history to return.
</ParamField>

<ParamField query="limit" type="number">
  Maximum number of usage records to return.
</ParamField>

<CodeGroup>
```bash curl
curl "https://api.usenaive.ai/v1/usage?days=30&limit=100" \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/usage",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    params={"days": 30, "limit": 100},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/usage?days=30&limit=100", {

});
const data = await response.json();
```
