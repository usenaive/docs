---
title: "Create Clips"
description: "Create short-form clips from a YouTube video."
api: "POST /v1/clips/create"
---

<ParamField body="youtube_url" type="string" required>
  YouTube video URL to create clips from.
</ParamField>

<ParamField body="template_name" type="string">
  Template to apply for styling and captions.
</ParamField>

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/clips/create \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "youtube_url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ",
    "template_name": "bold-captions"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.usenaive.ai/v1/clips/create",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    json={
        "youtube_url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ",
        "template_name": "bold-captions",
    },
)
data = response.json()
```
