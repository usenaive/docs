---
title: "Get Clip Status"
description: "Check clip generation status."
api: "GET /v1/clips/status/:projectId"
---

<ParamField path="projectId" type="string" required>
  Project ID from [Create Clips](/api-reference/creation/create-clips).
</ParamField>

<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/clips/status/sm_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/clips/status/sm_abc123",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

</CodeGroup>

<ResponseField name="status" type="string">
  `pending`, `processing`, `completed`, or `failed`.
</ResponseField>

<ResponseField name="clips" type="array">
  Generated clip objects (available when `completed`).
</ResponseField>

<ResponseField name="error" type="string">
  Error message (when `failed`).
</ResponseField>

```json Example Response
{
  "status": "completed",
  "clips": [
    {
      "id": "clip_1",
      "url": "https://cdn.usenaive.ai/clips/clip_1.mp4",
      "duration": 30,
      "title": "Best Moment - 0:45"
    }
  ]
}
```

Use [Edit Clip](/api-reference/creation/edit-clip) to modify generated clips.
