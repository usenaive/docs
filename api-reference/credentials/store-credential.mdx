---
title: "Store Credential"
description: "Securely store credentials for a service, encrypted at rest."
api: "POST /v1/credentials/store"
---

<ParamField body="service" type="string" required>
  Service name (e.g., `"github"`, `"stripe"`, `"openai"`).
</ParamField>

<ParamField body="credentials" type="object" required>
  Key-value pairs to store (e.g., `{"api_key": "sk-..."}`).
</ParamField>

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/credentials/store \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "service": "openai",
    "credentials": {"api_key": "sk-proj-abc123"}
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.usenaive.ai/v1/credentials/store",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    json={
        "service": "openai",
        "credentials": {"api_key": "sk-proj-abc123"},
    },
)
data = response.json()
```

</CodeGroup>

<ResponseField name="ok" type="boolean">
  Success flag.
</ResponseField>

<ResponseField name="ref" type="string">
  Credential reference for retrieval. Format: `paperclip://secrets/agent-credential/{agentId}/{service}`.
</ResponseField>

```json Example Response
{
  "ok": true,
  "ref": "paperclip://secrets/agent-credential/agent_1/openai"
}
```

Use the returned `ref` with [Get Credential](/api-reference/credentials/get-credential) to retrieve the stored values later.
