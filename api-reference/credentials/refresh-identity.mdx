---
title: "Refresh Identity"
description: "Re-generate identity scaffold files after credential or profile changes."
api: "POST /v1/credentials/refresh-identity"
---

No parameters required. The agent is resolved from the API key.

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/credentials/refresh-identity \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.post(
    "https://api.usenaive.ai/v1/credentials/refresh-identity",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/credentials/refresh-identity", {
  method: "POST",
  headers: { "Authorization": "Bearer nv_sk_your_key_here" },
});
const data = await response.json();
```

```javascript JavaScript
const response = await fetch("https://api.usenaive.ai/v1/credentials/refresh-identity", {
  method: "POST",
  headers: { "Authorization": "Bearer nv_sk_your_key_here" },
});
const data = await response.json();
```
</CodeGroup>

<ResponseField name="identity" type="string">
  Updated `identity.md` content.
</ResponseField>

<ResponseField name="accounts" type="string">
  Updated `accounts.md` with credential references.
</ResponseField>

<ResponseField name="profile" type="string">
  Updated `profile.md` with focus and context.
</ResponseField>

```json Example Response
{
  "identity": "# Agent Identity\n\nName: Writer\nEmail: writer@acme.usenaive.ai\n...",
  "accounts": "# Accounts\n\n- OpenAI: paperclip://secrets/agent-credential/agent_1/openai\n...",
  "profile": "# Profile\n\nFocus: Content creation and distribution\n..."
}
```

Call after storing new credentials, claiming a phone number, or registering a domain to keep identity files in sync.
