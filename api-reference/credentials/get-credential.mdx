---
title: "Get Credential"
description: "Retrieve stored credentials by reference. All access is audit-logged."
api: "POST /v1/credentials/resolve"
---

<ParamField body="ref" type="string" required>
  Credential reference from [Store Credential](/api-reference/credentials/store-credential). Format: `paperclip://secrets/{name}`.
</ParamField>

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/credentials/resolve \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"ref": "paperclip://secrets/agent-credential/agent_1/openai"}'
```
</CodeGroup>

<ResponseField name="secret_id" type="string">
  Secret ID.
</ResponseField>

<ResponseField name="name" type="string">
  Secret name.
</ResponseField>

<ResponseField name="credentials" type="object">
  The stored key-value pairs.
</ResponseField>

```json Example Response
{
  "secret_id": "sec_abc123",
  "name": "agent-credential/agent_1/openai",
  "credentials": {
    "api_key": "sk-proj-abc123"
  }
}
```

<Note>Every credential access is logged to the activity log for audit compliance.</Note>
