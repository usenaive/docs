---
title: "Get Contact Topics"
description: "Retrieve topic preferences for a contact."
api: "GET /v1/email/contacts/:contactId/topics"
---

<ParamField path="contactId" type="string" required>
  The contact ID.
</ParamField>

<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/email/contacts/ct_abc123/topics \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/email/contacts/ct_abc123/topics",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

