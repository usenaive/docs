<ParamField path="templateId" type="string" required>
  The template ID.
</ParamField>

<ParamField body="name" type="string">
  Updated template name.
</ParamField>

<ParamField body="subject" type="string">
  Updated subject line.
</ParamField>

<ParamField body="html" type="string">
  Updated HTML body.
</ParamField>

<ParamField body="text" type="string">
  Updated plain text body.
</ParamField>

<CodeGroup>
```bash curl
curl -X PATCH https://api.usenaive.ai/v1/email/templates/tpl_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"subject": "Welcome aboard, {{first_name}}!"}'
```

```python Python
import requests

response = requests.patch(
    "https://api.usenaive.ai/v1/email/templates/tpl_abc123",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    json={"subject": "Welcome aboard, {{first_name}}!"},
)
data = response.json()
```
