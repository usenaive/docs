---
title: "Delete Topic"
description: "Delete an email subscription topic."
api: "DELETE /v1/email/topics/:topicId"
---

<ParamField path="topicId" type="string" required>
  The topic ID to delete.
</ParamField>

<CodeGroup>
```bash curl
curl -X DELETE https://api.usenaive.ai/v1/email/topics/top_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.delete(
    "https://api.usenaive.ai/v1/email/topics/top_abc123",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/email/topics/top_abc123", {
```
