api: "GET /v1/email/contacts"
---

<ParamField query="limit" type="integer">
  Maximum number of results. Defaults to `25`.
</ParamField>

<ParamField query="after" type="string">
  Cursor for forward pagination.
</ParamField>

<ParamField query="before" type="string">
  Cursor for backward pagination.
</ParamField>

<ParamField query="segment_id" type="string">
  Filter contacts by segment ID.
</ParamField>

<CodeGroup>
```bash curl
curl -G https://api.usenaive.ai/v1/email/contacts \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -d limit=10
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/email/contacts",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    params={"limit": 10},
)
data = response.json()
```
