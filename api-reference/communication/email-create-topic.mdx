---
title: "Create Topic"
description: "Create a new email subscription topic."
api: "POST /v1/email/topics"
---

<ParamField body="name" type="string" required>
  Topic name (e.g. `Product Updates`, `Marketing`).
</ParamField>

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/email/topics \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"name": "Product Updates"}'
```

```python Python
import requests

response = requests.post(
    "https://api.usenaive.ai/v1/email/topics",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    json={"name": "Product Updates"},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/email/topics", {
```
