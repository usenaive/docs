---
title: "Remove Contact from Segment"
description: "Remove a contact from a segment."
api: "DELETE /v1/email/contacts/:contactId/segments/:segmentId"
---

<ParamField path="contactId" type="string" required>
  The contact ID.
</ParamField>

<ParamField path="segmentId" type="string" required>
  The segment ID to remove the contact from.
</ParamField>

<CodeGroup>
```bash curl
curl -X DELETE https://api.usenaive.ai/v1/email/contacts/ct_abc123/segments/seg_xyz789 \
  -H "Authorization: Bearer nv_sk_your_key_here"
```
