---
title: "Get Broadcast"
description: "Retrieve a single broadcast by ID."
api: "GET /v1/email/broadcasts/:broadcastId"
---

<ParamField path="broadcastId" type="string" required>
  The broadcast ID.
</ParamField>

<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/email/broadcasts/bc_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/email/broadcasts/bc_abc123",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/email/broadcasts/bc_abc123", {
  headers: { "Authorization": "Bearer nv_sk_your_key_here" },
});
const data = await response.json();
```
