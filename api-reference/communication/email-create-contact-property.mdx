<ParamField body="name" type="string" required>
  Property name (e.g. `company`, `plan_tier`).
</ParamField>

<ParamField body="type" type="string" required>
  Data type: `string`, `number`, `boolean`, or `date`.
</ParamField>

<ParamField body="description" type="string">
  Human-readable description of the property.
</ParamField>

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/email/contact-properties \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"name": "company", "type": "string", "description": "Company name"}'
```

```python Python
import requests

response = requests.post(
    "https://api.usenaive.ai/v1/email/contact-properties",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    json={"name": "company", "type": "string", "description": "Company name"},
)
data = response.json()
```
