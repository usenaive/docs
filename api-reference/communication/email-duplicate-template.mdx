---
title: "Duplicate Template"
description: "Create a copy of an existing email template."
api: "POST /v1/email/templates/:templateId/duplicate"
---

<ParamField path="templateId" type="string" required>
  The template ID to duplicate.
</ParamField>

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/email/templates/tpl_abc123/duplicate \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.post(
    "https://api.usenaive.ai/v1/email/templates/tpl_abc123/duplicate",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```
