---
title: "Get Sent Email Status"
description: "Retrieve delivery status, opens, and clicks for a sent email."
api: "GET /v1/email/sent/:emailId"
---

<ParamField path="emailId" type="string" required>
  The ID of the sent email.
</ParamField>

<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/email/sent/em_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/email/sent/em_abc123",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/email/sent/em_abc123", {

```

<ResponseField name="id" type="string">
  Email ID.
</ResponseField>

<ResponseField name="status" type="string">
  Delivery status: `queued`, `sent`, `delivered`, `bounced`, or `complained`.
</ResponseField>

<ResponseField name="opens" type="integer">
  Number of times the email was opened.
</ResponseField>

<ResponseField name="clicks" type="integer">
  Number of link clicks.
</ResponseField>
