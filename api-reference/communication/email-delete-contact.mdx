---
title: "Delete Contact"
description: "Permanently delete a contact."
api: "DELETE /v1/email/contacts/:contactId"
---

<ParamField path="contactId" type="string" required>
  The contact ID to delete.
</ParamField>

<CodeGroup>
```bash curl
curl -X DELETE https://api.usenaive.ai/v1/email/contacts/ct_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.delete(
    "https://api.usenaive.ai/v1/email/contacts/ct_abc123",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/email/contacts/ct_abc123", {
```
