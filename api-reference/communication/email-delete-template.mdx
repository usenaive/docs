---
title: "Delete Template"
description: "Delete an email template."
api: "DELETE /v1/email/templates/:templateId"
---

<ParamField path="templateId" type="string" required>
  The template ID to delete.
</ParamField>

<CodeGroup>
```bash curl
curl -X DELETE https://api.usenaive.ai/v1/email/templates/tpl_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.delete(
    "https://api.usenaive.ai/v1/email/templates/tpl_abc123",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/email/templates/tpl_abc123", {
```
