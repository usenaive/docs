<CodeGroup>
```bash curl
curl -X PATCH https://api.usenaive.ai/v1/email/contact-properties/prop_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"description": "Company or organization name"}'
```

```python Python
import requests

response = requests.patch(
    "https://api.usenaive.ai/v1/email/contact-properties/prop_abc123",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    json={"description": "Company or organization name"},
)
data = response.json()
```
