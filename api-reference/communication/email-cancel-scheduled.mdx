---
title: "Cancel Scheduled Email"
description: "Cancel a previously scheduled email before it is sent."
api: "POST /v1/email/sent/:emailId/cancel"
---

<ParamField path="emailId" type="string" required>
  The ID of the scheduled email to cancel.
</ParamField>

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/email/sent/em_abc123/cancel \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.post(
    "https://api.usenaive.ai/v1/email/sent/em_abc123/cancel",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

<ResponseField name="id" type="string">
  The cancelled email ID.
</ResponseField>

<ResponseField name="status" type="string">
  Updated status, will be `cancelled`.
</ResponseField>
