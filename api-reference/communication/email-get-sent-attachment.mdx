---
title: "Get Email Attachment"
description: "Retrieve a specific attachment from a sent email."
api: "GET /v1/email/sent/:emailId/attachments/:attachmentId"
---

<ParamField path="emailId" type="string" required>
  The ID of the sent email.
</ParamField>

<ParamField path="attachmentId" type="string" required>
  The ID of the attachment to retrieve.
</ParamField>

<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/email/sent/em_abc123/attachments/att_xyz789 \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/email/sent/em_abc123/attachments/att_xyz789",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/email/sent/em_abc123/attachments/att_xyz789", {
  headers: { "Authorization": "Bearer nv_sk_your_key_here" },
});
const data = await response.json();
```

```javascript JavaScript
const response = await fetch("https://api.usenaive.ai/v1/email/sent/em_abc123/attachments/att_xyz789", {

const data = await response.json();
```
</CodeGroup>

<ResponseField name="id" type="string">
  Attachment ID.
</ResponseField>

<ResponseField name="filename" type="string">
  Original filename.
</ResponseField>

<ResponseField name="content" type="string">
  Base64-encoded attachment content.
</ResponseField>

<ResponseField name="content_type" type="string">
  MIME type of the attachment.
</ResponseField>

```json Example Response
{
  "id": "att_xyz789",
  "filename": "report.pdf",
  "content": "JVBERi0xLjQK...",
  "content_type": "application/pdf"
}
```

Only emails sent by your agent are accessible. Returns 404 for emails belonging to other agents.
