---
title: "Add Contact to Segment"
description: "Add a contact to a segment."
api: "POST /v1/email/contacts/:contactId/segments"
---

<ParamField path="contactId" type="string" required>
  The contact ID.
</ParamField>

<ParamField body="segment_id" type="string" required>
  The segment ID to add the contact to.
</ParamField>

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/email/contacts/ct_abc123/segments \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"segment_id": "seg_xyz789"}'
```

```python Python
import requests

response = requests.post(
    "https://api.usenaive.ai/v1/email/contacts/ct_abc123/segments",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    json={"segment_id": "seg_xyz789"},
)
data = response.json()
```
