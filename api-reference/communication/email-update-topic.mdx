---
title: "Update Topic"
description: "Update an email subscription topic."
api: "PATCH /v1/email/topics/:topicId"
---

<ParamField path="topicId" type="string" required>
  The topic ID.
</ParamField>

<ParamField body="name" type="string">
  Updated topic name.
</ParamField>

<CodeGroup>
```bash curl
curl -X PATCH https://api.usenaive.ai/v1/email/topics/top_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"name": "Product News & Updates"}'
```

```python Python
import requests

response = requests.patch(
    "https://api.usenaive.ai/v1/email/topics/top_abc123",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    json={"name": "Product News & Updates"},
)
data = response.json()
```
