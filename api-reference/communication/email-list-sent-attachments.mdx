---
title: "List Email Attachments"
description: "List attachments from a sent email."
api: "GET /v1/email/sent/:emailId/attachments"
---

<ParamField path="emailId" type="string" required>
  The ID of the sent email.
</ParamField>

<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/email/sent/em_abc123/attachments \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/email/sent/em_abc123/attachments",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/email/sent/em_abc123/attachments", {
  headers: { "Authorization": "Bearer nv_sk_your_key_here" },
});
const data = await response.json();
```

```javascript JavaScript
const response = await fetch("https://api.usenaive.ai/v1/email/sent/em_abc123/attachments", {
  headers: { "Authorization": "Bearer nv_sk_your_key_here" },
});
const data = await response.json();
```
</CodeGroup>

<ResponseField name="data" type="array">
  List of attachment objects with `id`, `filename`, and `content_type`.
</ResponseField>

```json Example Response
{
  "data": [
    {
      "id": "att_xyz789",
      "filename": "report.pdf",
      "content_type": "application/pdf"
    }
  ]
}
```

Only emails sent by your agent are accessible. Returns 404 for emails belonging to other agents.
