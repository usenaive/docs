---
title: "Get Contact"
description: "Retrieve a single contact by ID."
api: "GET /v1/email/contacts/:contactId"
---

<ParamField path="contactId" type="string" required>
  The contact ID.
</ParamField>

<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/email/contacts/ct_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/email/contacts/ct_abc123",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/email/contacts/ct_abc123", {
```
