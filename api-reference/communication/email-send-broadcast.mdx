---
title: "Send Broadcast"
description: "Send or schedule a broadcast for delivery."
api: "POST /v1/email/broadcasts/:broadcastId/send"
---

<ParamField path="broadcastId" type="string" required>
  The broadcast ID to send.
</ParamField>

<ParamField body="scheduled_at" type="string">
  ISO 8601 datetime to schedule the broadcast. Omit to send immediately.
</ParamField>

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/email/broadcasts/bc_abc123/send \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"scheduled_at": "2026-05-01T09:00:00Z"}'
```

```python Python
import requests

response = requests.post(
    "https://api.usenaive.ai/v1/email/broadcasts/bc_abc123/send",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    json={"scheduled_at": "2026-05-01T09:00:00Z"},
)
data = response.json()
```
