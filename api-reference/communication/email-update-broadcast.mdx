<CodeGroup>
```bash curl
curl -X PATCH https://api.usenaive.ai/v1/email/broadcasts/bc_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"subject": "April Newsletter — Updated"}'
```

```python Python
import requests

response = requests.patch(
    "https://api.usenaive.ai/v1/email/broadcasts/bc_abc123",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    json={"subject": "April Newsletter — Updated"},
)
data = response.json()
```
