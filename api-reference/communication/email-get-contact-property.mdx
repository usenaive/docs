description: "Retrieve a single contact property definition."
api: "GET /v1/email/contact-properties/:propertyId"
---

<ParamField path="propertyId" type="string" required>
  The property ID.
</ParamField>

<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/email/contact-properties/prop_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/email/contact-properties/prop_abc123",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```
