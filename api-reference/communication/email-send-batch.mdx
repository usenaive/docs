---
title: "Send Batch Emails"
description: "Send multiple emails in a single API call."
api: "POST /v1/email/batch"
---

<ParamField body="emails" type="array" required>
  Array of email objects. Each must include `from`, `to`, `subject`, and `text` or `html`.
</ParamField>

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/email/batch \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "emails": [
      {"from": "agent@acme.com", "to": "a@example.com", "subject": "Hello A", "text": "Hi A!"},
      {"from": "agent@acme.com", "to": "b@example.com", "subject": "Hello B", "text": "Hi B!"}
    ]
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.usenaive.ai/v1/email/batch",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    json={
        "emails": [
            {"from": "agent@acme.com", "to": "a@example.com", "subject": "Hello A", "text": "Hi A!"},
            {"from": "agent@acme.com", "to": "b@example.com", "subject": "Hello B", "text": "Hi B!"},
        ]
    },
)
data = response.json()
```
