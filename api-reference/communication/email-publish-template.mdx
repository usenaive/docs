---
title: "Publish Template"
description: "Publish a draft template, making it available for use in emails and broadcasts."
api: "POST /v1/email/templates/:templateId/publish"
---

<ParamField path="templateId" type="string" required>
  The template ID to publish.
</ParamField>

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/email/templates/tpl_abc123/publish \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.post(
    "https://api.usenaive.ai/v1/email/templates/tpl_abc123/publish",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

<ResponseField name="id" type="string">
  Template ID.
</ResponseField>

  Updated status, will be `published`.
</ResponseField>
