---
title: "List Sent Emails"
description: "Retrieve a paginated list of sent emails."
api: "GET /v1/email/sent"
---

<ParamField query="limit" type="integer">
  Maximum number of results to return. Defaults to `25`.
</ParamField>

<ParamField query="after" type="string">
  Cursor for forward pagination. Use the `id` of the last item from the previous page.
</ParamField>

<ParamField query="before" type="string">
  Cursor for backward pagination.
</ParamField>

<CodeGroup>
```bash curl
curl -G https://api.usenaive.ai/v1/email/sent \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -d limit=10
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/email/sent",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    params={"limit": 10},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/email/sent?limit=10", {
  headers: { "Authorization": "Bearer nv_sk_your_key_here" },
});

```
