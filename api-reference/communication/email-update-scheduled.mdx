---
title: "Update Scheduled Email"
description: "Update the scheduled time of a pending email."
api: "PATCH /v1/email/sent/:emailId"
---

<ParamField path="emailId" type="string" required>
  The ID of the scheduled email to update.
</ParamField>

<ParamField body="scheduled_at" type="string">
  New ISO 8601 datetime for when the email should be sent.
</ParamField>

<CodeGroup>
```bash curl
curl -X PATCH https://api.usenaive.ai/v1/email/sent/em_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"scheduled_at": "2026-05-01T09:00:00Z"}'
```

```python Python
import requests

response = requests.patch(
    "https://api.usenaive.ai/v1/email/sent/em_abc123",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    json={"scheduled_at": "2026-05-01T09:00:00Z"},
)
data = response.json()
```
