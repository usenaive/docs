---

<ParamField path="contactId" type="string" required>
  The contact ID.
</ParamField>

<ParamField body="first_name" type="string">
  Updated first name.
</ParamField>

<ParamField body="last_name" type="string">
  Updated last name.
</ParamField>

<ParamField body="unsubscribed" type="boolean">
  Set unsubscribe status.
</ParamField>

<ParamField body="properties" type="object">
  Updated custom properties.
</ParamField>

<CodeGroup>
```bash curl
curl -X PATCH https://api.usenaive.ai/v1/email/contacts/ct_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"first_name": "Stephen", "properties": {"company": "Acme Corp"}}'
```

```python Python
import requests

response = requests.patch(
    "https://api.usenaive.ai/v1/email/contacts/ct_abc123",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    json={"first_name": "Stephen", "properties": {"company": "Acme Corp"}},
)
data = response.json()
```
