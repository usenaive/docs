---
title: "Create Template"
description: "Create a new email template."
api: "POST /v1/email/templates"
---

<ParamField body="name" type="string" required>
  Template name.
</ParamField>

<ParamField body="subject" type="string" required>
  Default subject line.
</ParamField>

<ParamField body="html" type="string">
  HTML template body. Supports variables like `{{first_name}}`.
</ParamField>

<ParamField body="text" type="string">
  Plain text template body.
</ParamField>

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/email/templates \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Welcome Email",
    "subject": "Welcome, {{first_name}}!",
    "html": "<h1>Welcome!</h1><p>Thanks for signing up, {{first_name}}.</p>"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.usenaive.ai/v1/email/templates",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    json={
        "name": "Welcome Email",
        "subject": "Welcome, {{first_name}}!",
        "html": "<h1>Welcome!</h1><p>Thanks for signing up, {{first_name}}.</p>",
    },
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/email/templates", {

});
const data = await response.json();
```
