title: "Create Segment"
description: "Create a new contact segment."
api: "POST /v1/email/segments"
---

<ParamField body="name" type="string" required>
  Segment name.
</ParamField>

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/email/segments \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"name": "Newsletter Subscribers"}'
```

```python Python
import requests

response = requests.post(
    "https://api.usenaive.ai/v1/email/segments",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    json={"name": "Newsletter Subscribers"},
)
data = response.json()
```
