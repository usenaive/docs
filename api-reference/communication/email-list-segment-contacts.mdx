---
title: "List Segment Contacts"
description: "Retrieve all contacts in a segment."
api: "GET /v1/email/segments/:segmentId/contacts"
---

<ParamField path="segmentId" type="string" required>
  The segment ID.
</ParamField>

<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/email/segments/seg_abc123/contacts \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/email/segments/seg_abc123/contacts",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/email/segments/seg_abc123/contacts", {
```
