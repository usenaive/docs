---
title: "Delete Segment"
description: "Delete a contact segment."
api: "DELETE /v1/email/segments/:segmentId"
---

<ParamField path="segmentId" type="string" required>
  The segment ID to delete.
</ParamField>

<CodeGroup>
```bash curl
curl -X DELETE https://api.usenaive.ai/v1/email/segments/seg_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.delete(
    "https://api.usenaive.ai/v1/email/segments/seg_abc123",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```
