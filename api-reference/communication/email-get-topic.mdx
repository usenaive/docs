---
title: "Get Topic"
description: "Retrieve a single topic by ID."
api: "GET /v1/email/topics/:topicId"
---

<ParamField path="topicId" type="string" required>
  The topic ID.
</ParamField>

<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/email/topics/top_abc123 \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/email/topics/top_abc123",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/email/topics/top_abc123", {
```
