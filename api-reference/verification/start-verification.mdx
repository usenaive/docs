---
title: "Start Verification"
description: "Initiate identity verification for a formation member. Returns a hosted URL the user opens in their browser."
api: "POST /v1/verification/start"
---

<ParamField body="formation_draft_id" type="string" required>
  The formation draft this verification is associated with.
</ParamField>

<ParamField body="member_id" type="string" required>
  Unique identifier of the member to verify.
</ParamField>

<ParamField body="email" type="string">
  Email address to pre-fill on the verification form.
</ParamField>

<ParamField body="first_name" type="string">
  First name to pre-fill on the verification form.
</ParamField>

<ParamField body="last_name" type="string">
  Last name to pre-fill on the verification form.
</ParamField>

<ParamField body="is_us_resident" type="boolean" required>
  Whether the member is a US resident. Determines which verification playbook is used.
</ParamField>

<Note>
  The returned `hosted_url` should be presented to the user to complete verification in their browser. The agent never interacts with PII directly.
</Note>

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/verification/start \
  -H "Authorization: Bearer nv_sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "formation_draft_id": "fd_abc123",
    "member_id": "mem_xyz789",
    "email": "jane@example.com",
    "first_name": "Jane",
    "last_name": "Doe",
    "is_us_resident": true
  }'
```

```python Python
import requests

        "member_id": "mem_xyz789",
        "email": "jane@example.com",
        "first_name": "Jane",
        "last_name": "Doe",
        "is_us_resident": True,
    },
)
data = response.json()
```

  ISO 8601 timestamp when the hosted URL expires.
</ResponseField>

<ResponseField name="playbook_id" type="string">
  The verification playbook selected based on residency.
</ResponseField>

<ResponseField name="member_id" type="string">
  The member ID this verification was started for.
