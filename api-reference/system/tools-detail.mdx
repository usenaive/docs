---
title: "Get Tool Detail"
description: "Get the full definition and parameter schema for a specific tool. No authentication required."
api: "GET /v1/tools/:toolName"
---

<Note>This endpoint is **unauthenticated** — no API key required.</Note>

<ParamField path="toolName" type="string" required>
  Tool name (e.g., `send_email`, `create_social_post`).
</ParamField>

