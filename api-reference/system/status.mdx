---
title: "Status"
description: "Get agent identity, credits, and available primitives."
api: "GET /v1/status"
---

No parameters required. Returns the identity and capabilities of the agent associated with the API key.

<CodeGroup>
```bash curl
curl https://api.usenaive.ai/v1/status \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/status",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/status", {
  headers: { "Authorization": "Bearer nv_sk_your_key_here" },
});
const data = await response.json();
```

```javascript JavaScript

```
