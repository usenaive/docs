---
title: "List Formation Documents"
description: "List all documents associated with a formation draft, such as articles of organization, EIN letters, and operating agreements."
api: "GET /v1/formation/drafts/{id}/documents"
---

<ParamField path="id" type="string" required>
  The `draft_id` returned from [Draft Formation](/api-reference/formation/draft-formation).
</ParamField>

<ParamField query="types" type="string">
  Comma-separated list of document types to filter (e.g., `"articles_of_organization,ein_letter"`). Omit to return all documents.
</ParamField>

<CodeGroup>
```bash curl
curl "https://api.usenaive.ai/v1/formation/drafts/fdr_abc123/documents?types=articles_of_organization" \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.usenaive.ai/v1/formation/drafts/fdr_abc123/documents",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
    params={"types": "articles_of_organization"},
)
data = response.json()
```
