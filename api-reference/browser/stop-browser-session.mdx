---
title: "Stop Browser Session"
description: "Stop a running browser session for the agent."
api: "POST /v1/browser/stop"
---

No parameters required.

<CodeGroup>
```bash curl
curl -X POST https://api.usenaive.ai/v1/browser/stop \
  -H "Authorization: Bearer nv_sk_your_key_here"
```

```python Python
import requests

response = requests.post(
    "https://api.usenaive.ai/v1/browser/stop",
    headers={"Authorization": "Bearer nv_sk_your_key_here"},
)
data = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.usenaive.ai/v1/browser/stop", {
  method: "POST",

});

```
