Video is the **creation primitive** for moving content. Naive exposes two engines behind a consistent interface: **AI avatar videos** where a digital presenter speaks the employee's script, and **clip extraction** for pulling viral-ready short-form clips from existing YouTube content. A marketer or designer employee with the video whitelist can use either without juggling separate accounts — billing, storage, and credentials all flow through the company.

## Tools

| Tool | Type | Description |
|------|------|-------------|
| `list_video_styles` | Extended | Browse available avatar styles |
| `create_video` | Extended | Create an AI avatar video from a text prompt |
| `get_video_session` | Extended | Poll video generation status and progress |
| `send_video_message` | Extended | Send follow-up instructions to an active session |
| `get_video` | Extended | Get the final video URL for a completed video |
| `create_clips` | Extended | Generate short-form clips from a YouTube URL |
| `get_clip_status` | Extended | Check clip generation progress |
| `edit_clip` | Extended | Edit a previously generated clip |

## AI Avatar Videos

The video engine creates videos with AI-generated avatars that speak your text with realistic lip-sync and natural expressions.

### Workflow

<Steps>
  <Step title="Browse avatar styles">
    ```bash
    curl "https://api.usenaive.ai/v1/video/styles?tag=business" \
      -H "Authorization: Bearer nv_sk_your_key"
    ```

    Response:
    ```json
    {
      "ok": true,
      "styles": [
        { "id": "style_abc", "name": "Professional Presenter", "tag": "business", "preview": "https://..." },
        { "id": "style_def", "name": "Casual Speaker", "tag": "casual", "preview": "https://..." }
      ]
    }
    ```

    Filter by tags like `business`, `casual`, or omit for all styles.
  </Step>

  <Step title="Create the video">
    ```bash
    curl -X POST https://api.usenaive.ai/v1/video/create \
      -H "Authorization: Bearer nv_sk_your_key" \
      -H "Content-Type: application/json" \
      -d '{
        "prompt": "Welcome to Acme Corp! In this video, I will walk you through our three core products and show you how they can transform your workflow."
      }'
    ```

    Response:
    ```json
    {
      "ok": true,
      "sessionId": "sess_abc123",
      "status": "pending",
      "callbackId": "cb_xyz"
    }
    ```

    Video generation is asynchronous — it takes a few minutes.
  </Step>

  <Step title="Poll for completion">
    ```bash
    curl https://api.usenaive.ai/v1/video/session/sess_abc123 \
      -H "Authorization: Bearer nv_sk_your_key"
    ```

    Response:
    ```json
    {
      "ok": true,
      "sessionId": "sess_abc123",
      "status": "processing",
      "progress": 45,
      "title": "Acme Corp Welcome Video"
    }
    ```

    Poll every 15-30 seconds until `status` is `completed`. The `progress` field shows percentage.
  </Step>

  <Step title="Iterate with follow-up messages (optional)">
    Send additional instructions to refine the video while the session is active:

    ```bash
    curl -X POST https://api.usenaive.ai/v1/video/session/sess_abc123 \
      -H "Authorization: Bearer nv_sk_your_key" \
      -H "Content-Type: application/json" \
      -d '{ "message": "Add a call-to-action at the end: Visit acme.com to get started" }'
    ```
  </Step>

  <Step title="Get the final video">
    Once the session status shows `completed` with a `videoId`:

    ```bash
    curl https://api.usenaive.ai/v1/video/video/vid_abc123 \
      -H "Authorization: Bearer nv_sk_your_key"
    ```

    Response:
    ```json
    {
      "ok": true,
      "status": "completed",
      "videoUrl": "https://cdn.usenaive.ai/video/abc123.mp4",
      "duration": 45,
      "thumbnailUrl": "https://cdn.usenaive.ai/video/thumb/abc123.jpg"
    }
    ```

    Use `videoUrl` to distribute the video — post it to social media, embed it on your site, or download it.
  </Step>
</Steps>

### Avatar Video Tools

| Tool | Method | Description |
|------|--------|-------------|
| `list_video_styles` | GET | Browse available avatar styles |
| `create_video` | POST | Create a new video from a text prompt |
| `get_video_session` | GET | Poll session status and progress |
| `send_video_message` | POST | Send follow-up instructions to an active session |
| `get_video` | GET | Get the final video URL (credits billed on first retrieval) |

### Cost

Each avatar video costs approximately **5-10 credits**. Credits are billed when you first retrieve the final video with `get_video`.

### Error Recovery

| Error | Recovery |
|-------|----------|
| `insufficient_credits` | Top up credits at the developer portal |
| `session_not_found` | Session doesn't exist — start a new one |

## Clip Extraction

The clip engine takes a YouTube video URL and generates short-form clips optimized for social media — with auto-generated captions, smart cropping, and engaging formatting.

### Workflow

<Steps>
  <Step title="Create clips from a YouTube URL">
    ```bash
    curl -X POST https://api.usenaive.ai/v1/clips/clips \
      -H "Authorization: Bearer nv_sk_your_key" \
      -H "Content-Type: application/json" \
      -d '{ "youtube_url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ" }'
    ```

    Response:
    ```json
    {
      "ok": true,
      "projectId": "proj_abc123"
    }
    ```
  </Step>

  <Step title="Poll for completion">
    ```bash
    curl https://api.usenaive.ai/v1/clips/status/proj_abc123 \
      -H "Authorization: Bearer nv_sk_your_key"
    ```

    Response when complete:
    ```json
    {
      "status": "completed",
      "clips": [
        { "id": "clip_1", "title": "Key Insight", "duration": 30, "url": "https://..." },
        { "id": "clip_2", "title": "Best Moment", "duration": 45, "url": "https://..." }
      ]
    }
    ```
  </Step>

  <Step title="Edit clips (optional)">
    Trim, adjust captions, or modify formatting:

    ```bash
    curl -X POST https://api.usenaive.ai/v1/clips/edit \
      -H "Authorization: Bearer nv_sk_your_key" \
      -H "Content-Type: application/json" \
      -d '{ "clip_id": "clip_1", "trim_start": 5, "trim_end": 25 }'
    ```
  </Step>

  <Step title="Distribute">
    Use the clip URLs to post directly to social media:

    ```bash
    curl -X POST https://api.usenaive.ai/v1/social/posts \
      -H "Authorization: Bearer nv_sk_your_key" \
      -H "Content-Type: application/json" \
      -d '{
        "content": "The key insight from our latest episode",
        "platforms": ["TIKTOK", "INSTAGRAM", "YOUTUBE"],
        "media_urls": ["https://cdn.usenaive.ai/clips/clip_1.mp4"],
        "youtube_type": "SHORT"
      }'
    ```
  </Step>
</Steps>

## Choosing Between Avatar Videos and Clip Extraction

| | Avatar Video | Clip Extraction |
|---|--------|---------|
| **Input** | Text script | YouTube URL |
| **Output** | AI avatar video | Short-form clips |
| **Best for** | Product demos, announcements, tutorials | Repurposing long-form content |
| **Duration** | Custom (based on script length) | Auto-generated clips (15-60s) |
| **Cost** | ~5-10 credits | Per-project cost |

<Tip>
Combine both: create a long-form avatar video, upload it to YouTube, then use clip extraction to generate short clips for TikTok and Instagram Reels.
</Tip>

## Typical Workflow

```
Agent receives task: "Create a product announcement video and clip it for social"
    │
    ├─ create_video({                          → Generate full video
    │    prompt: "Welcome to Acme 2.0..."
    │  })
    │   → sessionId: sess_abc123
    │
    ├─ get_video_session({ sessionId })        → Poll until complete
    │   → status: completed, videoId: vid_xyz
    │
    ├─ get_video({ videoId })                  → Get video URL
    │   → videoUrl: https://cdn.usenaive.ai/video/...
    │
    ├─ create_clips({                          → Extract clips for social
    │    youtube_url: "[uploaded video URL]"
    │  })
    │   → projectId: proj_def456
    │
    ├─ get_clip_status({ projectId })          → Poll until clips ready
    │   → 3 clips generated
    │
    └─ create_social_post({                    → Post clips to social
         media_urls: ["clip_url"],
         platforms: ["TIKTOK", "INSTAGRAM"]
       })
```
